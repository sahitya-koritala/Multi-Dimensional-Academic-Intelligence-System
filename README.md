# Multi-Dimensional-Academic-Intelligence-System
Day-8 Challenge of Python 

## Overview
This project analyzes student performance using multiple factors such as marks, attendance, and assignment scores instead of relying on marks alone. It generates student data, processes it using Python libraries, and provides meaningful insights about overall academic performance.

## Objectives 
- To evaluate student performance using multiple parameters
- To perform statistical analysis on student data
- To classify students into different performance categories
- To generate meaningful insights from the dataset

## What this program does
- Generates student data using random values
- Stores data using list, tuple, and dictionary
- Converts data into a Pandas DataFrame
- Classifies students into categories like At Risk, Average, Good, and Top Performer
- Calculates statistics like mean, median, and standard deviation
- Finds relation between marks and attendance
- Gives a final result about overall performance

## Technologies Used
- Python
- Pandas
- NumPy
- Random Module
- Math Module

## Data Structure
Each student has:
- ID
- Marks (0–100)
- Attendance (0–100)
- Assignment score (0–50)

## How to Run
1. Install required libraries:
pip install pandas numpy
2. Run the Python file:
python main.py
3. Enter your roll number when prompted

## Conclusion

This project demonstrates how Python can be used to analyze student performance in a structured and efficient way, providing deeper insights compared to traditional methods.
